# 결정 사항 (국어농장 v2)

기획서: `국어농장_홈페이지_프로그램_기획서_유료화_반영판.md`

## 결제와 권한
- 웹 카드 결제만 지원한다 (유료 회원 전환, 쇼핑몰 교재/교구 결제).
- 유료 회원은 월 구독 자동 갱신이며, 해지 시 만료일까지 유지 후 무료 전환한다.
- 게스트는 무료 학습 미리보기만 가능하며, 대결 모드는 로그인 필수다.
- 쿠폰/포인트/복수 결제수단/인앱 결제는 제외한다.

## 운영과 플래그
- 모든 기능은 배포 후 플래그로 노출 제어한다.
- 긴급 중지 대상은 대결/결제/업로드다.
- 일일 리셋과 시즌 집계 기준 시간대는 `Asia/Seoul` 00:00다.

## 시즌과 대결
- 시즌은 운영자가 시작/종료하며 기본 분기는 3개월이다.
- 대결 규칙/타임아웃/스테이크 범위는 기획서 JSON 기본값을 따른다.
- ELO를 사용하고 동일 레벨 우선 매칭을 적용한다.
- 승률왕 최소 참가 수는 20이며, 수수료는 0에서 시작한다.

## 학습과 경제
- 씨앗 지급은 영역별 고정, 하루 1회, 2개 지급 규칙을 따른다.
- 수확 제작은 일 30회, 비료 1개 사용 시 3배 수확이다.
- 재화 변동은 장부에 기록한다 (종류/수량/사유/연관 활동).

## 데이터와 마이그레이션
- 콘텐츠는 정적, 시도/전적 데이터는 사용자별 누적이다.
- v1 씨앗 단일 수치는 `wheat/oat/rice` 균등 분배, 잔여는 `wheat`에 합산한다.
- v1 학습 이력/점수/보상/유료 권한은 유지하며 최초 로그인 시 1회 마이그레이션한다.

## 관리자 범위
- 필수: 기관/학생/수강반, 과제 배정/현황, 학습 데이터 JSON 업로드·검증, 게시판 승인.
- 후순위: 고급 통계, 판매 대시보드, 리포트 자동화.

## Membership roles
- Roles: HQ_ADMIN, ORG_ADMIN, STUDENT, PARENT.
- Paid vs free students are determined by subscription status, not roles.
- Parent/student access is linked via `parent_student_links`.
