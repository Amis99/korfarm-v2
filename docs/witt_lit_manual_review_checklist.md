# 비트겐슈타인 문학 수동 검수 체크리스트

## 공통 원칙
- 스크립트 자동 출제 금지. 문항은 수동으로 작성한다.
- 지문은 원문 그대로 사용하되, 길면 한 덩어리 연속 구간만 발췌한다.
- 중간중간 생략 연결(앞뒤 조각 결합) 금지.
- 요약/의역/문체 변형 금지.

## 문항별 필수 점검
1. 단일 작품 지문 여부
- 지문에 한 작품만 남아 있는가?

2. 표식 제거 여부
- `(가)(나)(다)`, `[A][B]`, `㉠㉡`, `ⓐⓑ` 등 문제 표식을 제거했는가?

3. 길이 제한
- 지문이 500자 이하인가?

4. 원문 보존
- 맞춤법/어휘/어순 임의 수정 없이 원문을 유지했는가?

5. 수능형 문항 적합성
- 발문이 수능형(이해/감상/표현/화자 태도 등)인가?

6. 정답 유일성
- 선택지 5개 중 정답 1개만 지문 근거로 성립하는가?

7. 오답 타당성
- 오답 4개가 그럴듯하지만 지문 근거로 배제 가능한가?

## 반영 후 검증
- API 재조회로 `stem`, `passage`, `choices`, `answerId`를 확인한다.
- 위반 발견 시 즉시 해당 문항을 수동 재수정한다.
