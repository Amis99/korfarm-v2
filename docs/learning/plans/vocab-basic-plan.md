# 어휘 기본 학습 구현 계획 (VOCAB_BASIC)

## 1) 범위
- 어휘 기본 학습 3가지 유형을 한 학습 모듈(Worksheet 기반)에서 처리합니다.
  1. 낱말 → 뜻 고르기
  2. 뜻 → 낱말 고르기
  3. 용례 빈칸 → 낱말 고르기

## 2) 화면 구성
- A4 세로 시험지 비율 고정(최대 A4 실제 크기), 화면 폭에 맞춰 축소
- 헤더(두 줄)
  - 1줄: 학습명 / 학습 종류·레벨 / 영역·세부 영역
  - 2줄: 씨앗 아이콘 / 시간바(가로 확장)
- 본문
  - 시험지 배경 이미지 위에 문제 텍스트 배치
  - 문제는 상단부터 순차 진행
- 푸터
  - 좌측: 학습 종료 버튼
  - 우측: 진행바(현재 문제수/전체 문제수)

## 3) 문제 표시 규칙
- 공통
  - 문제(stem)는 본문에 표시
  - 모달에는 prompt + 선택지 표시
- 유형별
  - 낱말 → 뜻: stem에 낱말 제시, prompt는 “뜻을 고르세요”
  - 뜻 → 낱말: stem에 뜻 제시, prompt는 “낱말을 고르세요”
  - 용례 빈칸: template/blank 구조 사용

## 4) 정오답 처리
- 정답: 문제 텍스트에 동그라미 표시(○), 시간바 보상(기본 +20초)
- 오답: 문제 텍스트에 사선 표시(／), 시간바 패널티(기본 -20초)
- 기본 동작: 오답이어도 다음 문제로 진행
- 옵션: `requireCorrect: true`인 문제는 정답을 맞힐 때까지 반복

## 5) 시간/씨앗 규칙
- 기본 제한시간 3분
- 0초가 되면 씨앗 1개 차감, 씨앗 0이면 실패 종료
- 정오답 시간 증감은 문제별 scoring에 따름

## 6) 데이터 입력(JSON 표준)
- contentType: `VOCAB_BASIC`
- area: `VOCAB`
- subArea: `BASIC`
- payload.questions[] 사용
- question.type
  - `MULTI_CHOICE` 또는 `FILL_BLANKS`
- `questionKind`로 3가지 유형 표시

## 7) 관리자 입력 흐름
- 관리자 페이지에서 “어휘 기본 학습 템플릿”을 불러온 뒤
  - 제목, 레벨, 난이도, 질문/선택지/정답을 채움
  - scoring 값을 필요에 따라 조정
