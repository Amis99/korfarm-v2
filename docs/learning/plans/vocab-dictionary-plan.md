# 사전 학습 구현 계획 (VOCAB_DICTIONARY)

## 1) 범위
- 사전 학습은 3개 유형으로 구성
  1. 사전적 의미를 보고 낱말 맞히기
  2. 낱말을 보고 사전적 의미를 완성하기 (빈칸 구절 단계별 선택)
  3. 다의어/동음이의어 예문 보고 뜻 맞히기

## 2) 화면 구성
- 시험지 A4 배경 위에 사전 항목 박스를 미리 출력
- 각 사전 항목은 테두리 박스로 묶어 표시
- 낱말/빈칸이 필요한 경우 사전 항목 내부에서 빈칸(____)으로 표현
- 문제 진행 시 빈칸(또는 선택 구절)만 하이라이트

## 3) 상호작용 흐름
- 유형 1: 사전적 의미(정의) + 예문을 제시하고 낱말은 빈칸 처리
  - 낱말 빈칸이 하이라이트되고 모달에서 낱말 선택
- 유형 2: 낱말은 그대로 제시, 사전 정의를 빈칸 구절로 분해
  - 빈칸(____) 위치에 대해 모달이 순차적으로 떠서 선택
  - 선택 결과가 해당 빈칸에 채워짐
- 유형 3: 예문을 제시하고 낱말은 빈칸 처리
  - 낱말 빈칸이 하이라이트되고 모달에서 뜻 선택

## 4) 데이터 모델
- contentType: `VOCAB_DICTIONARY`
- payload.words[]
  - wordId, headword, pos(품사), sense, definitionParts[]
  - exampleSentences[] (다의어/동음이의어용)
- questions[]
  - questionKind: DICT_MEANING_TO_WORD / WORD_TO_DICT_FILL / HOMONYM_EXAMPLE
  - linkWordId로 단어 박스와 연결
  - headwordBlank로 낱말 빈칸 여부 제어 (기본값은 유형별 자동)
