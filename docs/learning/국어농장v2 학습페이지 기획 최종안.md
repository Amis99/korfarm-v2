# 학습 종류

1. 어휘 학습  
   1. 어휘 기본 학습  
      1. 제시된 낱말을 보고 뜻을 모달에서 고르는 학습  
      2. 제시된 뜻을 보고 낱말을 모달에서 고르는 학스  
      3. 용례의 빈칸에 들어갈 낱말을 모달에서 고르는 학습  
   2. 사전(dictionary) 학습  
      1. 사전적 의미를 보고 낱말 맞히기  
      2. 낱말을 보고 사전적 의미를 완성하기  
         1. 낱말이 화면에 제시되고 사전적 의미 부분들에 빈칸이 있고 빈칸마다 순차적으로 모달이 떠 4지 선다로 사전적 의미의 한 구절 한 구절씩 맞혀 채워나가는 학습  
         2. 먹다 : \[동사\] 「…을」음식 따위를 입을 통하여 배 속에 들여보내다.  
            를 예로 들면   
            먹다 : \[\_\_\_\] 「\_\_\_」\_\_\_ 따위를 \_\_\_을 통하여 \_\_\_에 \_\_\_\_\_\_다.  
            와 같이 6문제가 순차적으로 모달로 나옴  
      3. 다의어 또는 동음이의어의 경우 예문을 보고 먹다1의 뜻, 먹다2의 뜻, 먹다3의 뜻, 먹다4의 뜻을 모달에서 고르는 학습  
2. 독해 연습  
   1. 비문학 독해 연습  
      1. 정독 훈련  
         1. 지문 1단 구성  
         2. 기본 진행 : 문장이 순차적으로 하이라이트된 후 모달에 질문과 선택지가 뜸  
         3. 접속어 처리 : 접속어가 하이라이트된 후 모달에 이어질 내용을 추측하게 함  
         4. 치환하기 : 대용 표현이 나왔거나 지문의 앞에서 정의된 내용이 있거나, 일반적 진술 뒤에 나온 사례(혹은 반대 순서)의 경우 해당 내용이 하이라이트된 후 모달에 바꿔 쓸 수 있는 말을 고르게 함  
         5. 한 문단의 모든 문장을 다 읽은 후 : 해당 문단을 모두 하이라이트한 후 모달에서 문단 중심 내용을 고르게 함  
         6. 정답 시 \+20초, 오답 시 \-20초   
         7. 오답 시 선택한 선택지를 지우고 남은 선택지로 모달에서 풀이 지속  
         8. 정답을 맞혀야 다음 하이라이트로 넘어감  
         9. 훈련 완료 후 씨앗 지급  
      2. 복기 학습 : 정독 훈련 후 진행  
         1. 상단의 지문 박스(텍스트는 없고 비어 있음) \+ 하단의 지문 요약 카드 5\~10장  
         2. 하단의 지문 요약 카드를 상단의 원하는 위치로 이동  
         3. 상단의 카드끼리 순서 이동 가능  
         4. 이동 시 애니메이션  
         5. 푸터에 제출 버튼  
         6. 정확한 순서일 시 성공 안내 및 씨앗 지급  
         7. 틀렸을 경우 정확한 자리의 카드들을 표시한 후 학습에서 제공할 씨앗 하나 삭제 \- 씨앗이 0이 되면 학습 종료  
      3. 확인 학습 : 복기 학습 후 진행  
         1. 상단의 지문 박스(80% 높이/스크롤 가능) \+ 하단의 문제(20% 높이)  
         2. 하단의 문제에 해당하는 표현을 클릭하면 정답  
         3. 정답이 ‘사실에 대한 지배자’일 경우   
            1. 지문에서 해당 표현의 위치를 모두 파악한 후  
            2. 해당 표현들 중 한 곳에서 해당 표현의 첫 글자(사)와 마지막 글자(자) 사이를 클릭하면 (사, 실, 에, 빈칸, 대, 한, 빈칸, 지, 배, 자 중 한 곳에)  
            3. 정답으로 처리  
            4. 단순히 같은 글자라고 정답으로 처리해선 안 됨. a.에서 파악한 위치 안에서 클릭이 되어야 함. (‘사람들’이란 표현의 ‘사’를 클릭하면 오답)  
         4. 정답 시 \+30초, 오답 시 \-30초  
         5. 오답 시 지문에서 정답 표현을 하이라이트로 보여준 후 클릭(터치) 후 다음 문제로 진행(시간바는 계속 진행 중)  
         6. 훈련 완료 후 씨앗 지급

3. 내용 숙지 학습  
   1. 업로드한 PDF(A4 세로 사이즈)를 보여준 후 푸터의 학습 시작 버튼을 누르면 학습 시작  
   2. 해당 PDF에 해당하는 학습 문제들은 미리 관리자가 json형식으로 입력해둠  
   3. 학습 시작이 되면 PDF 화면은 사라지고, 시험지 배경에 문제들이 뜸  
   4. 각 문제들마다 순차적으로 모달이 뜨고  
      1. 빈칸 채우기 : 빈칸 문제인 경우 빈칸이 포함된 문제에 하이라이트되고 모달이 뜨면 빈칸에 들어갈 말을 선택지 중 골라야 함  
      2. OX 퀴즈 : OX 문제인 경우 해당 문장에 하이라이트되고 모달이 뜨면 O 또는 X 카드 중 하나를 고르게 해야 함  
      3. 단답형 문제 : 단답형 문제인 경우 해당 문제에 하이라이트되고 모달이 뜨면 선택지 중 답을 골라야 함  
      4. 서술형 문제 : 서술형 문제인 경우 해당 문제에 하이라이트되고 모달이 뜨면 원고지에 어절 수만큼 하이라이트 후 하이라이트된 곳에 들어갈 말을 골라 나가면서 답을 완성해야 함.  
         1. 답이 ‘카는 사실을 '과거에 대한 사실', '역사상의 사실', '역사적 사실'로 구분했다.’라면  
            \_ \_  \_ \_ \_  ‘ \_ \_ \_  \_ \_  \_ \_ ‘ , ‘ \_ \_ \_ \_  \_ \_ ‘ , ‘ \_ \_ \_  \_ \_ ‘ \_  \_ \_ \_ \_  
            처럼 빈 칸을 주고  
            처음에는 ‘카는 / 랑케는’ 중에 고르게 한 후, ‘해석을 / 사실을’을 고르게 하고\~ 순차적으로 어절을 다 채워서 ‘구분했다 / 조명했다’를 고른 후 마무리되는 형식이야  
      5. 각각의 문제는 정답을 맞힐 때까지가 아니라 틀리면 바로 학습 페이지(시험지 디자인의)에 하이라이트되었던 문장이나 문제에 색연필로 틀림 표시(사선 그음)으로 표시하고 시간바의 시간을 \-30초 삭제할 것  
      6. 정답을 맞힐 때마다 시간바의 시간을 \+30초 채우고, 하이라이트되었던 문장이나 문제에 색연필로 맞음 표시(동그라미 채점)을 표시할 것  
      7. 푸터에 PDF보기 버튼  
         1. PDF 보기를 누르면 PDF를 볼 수 있음  
         2. 시간바는 그대로 계속 감  
         3. 보기 종료를 누르면 다시 학습 페이지로 가서 이어서 함

   1. 시험지 배경에 문제들이 떠 있음  
   2. 문제들이 하이라이트되고 데이터셋의 문제들이 모달에 나옴  
8. 서술형 연습  
   1. 시험지 배경에 지문과 문제가 떠 있음  
   2. 문제들이 하이라이트되고 모달이 뜸  
   3. 모달이 뜨면 원고지에 어절 수만큼 하이라이트 후 하이라이트된 곳에 들어갈 말을 골라 나가면서 답을 완성해야 함.  
      1. 답이 ‘카는 사실을 '과거에 대한 사실', '역사상의 사실', '역사적 사실'로 구분했다.’라면  
         \_ \_  \_ \_ \_  ‘ \_ \_ \_  \_ \_  \_ \_ ‘ , ‘ \_ \_ \_ \_  \_ \_ ‘ , ‘ \_ \_ \_  \_ \_ ‘ \_  \_ \_ \_ \_  
         처럼 빈 칸을 주고  
         처음에는 첫 어절에 ‘카는 / 랑케는’ 중에 고르게 한 후, ‘해석을 / 사실을’을 고르게 하고\~ 순차적으로 어절을 다 채워서 ‘구분했다 / 조명했다’를 고른 후 마무리되는 형식이야  
9. 선택지 판별 연습  
   1. 시험지 배경에 지문과 문제가 떠 있음  
   2. 문제의 선택지를 클릭(터치)하면 각 선택지가 데이터셋에 설정된 형태의 명제들로 분해되어 순차적으로 모달에 뜸  
   3. 각각의 명제들에 대한 근거를 지문에서 찾아 클릭하면 정답일 경우 보상을 주고(+20초) 지문의 해당 구절에 하이라이트가 됨. 오답일 경우 시간 벌칙(-20초)을 주고 계속 진행  
   4. 정답일 경우 모달에 해당 명제 뒤에 OX 카드가 생성됨  
   5. OX가 정답일 경우 보상을 주고 다음 명제가 모달에 제시됨  
   6. c로 돌아가 반복 진행  
   7. 해당 선택지에 배정된 명제에 대해 모두 진행된 후  
   8. 해당 선택지가 정답인 선택지라면 다음 문제로 진행  
   9. 해당 선택지가 정답이 아닌 경우 해당 선택지에 O(적절하지 않은 것을 고르는 경우) 또는 X(적절한 선택지를 고르는 경우)를 연필 표시를 남긴 후 다른 선택지를 고르게 함  
   10. b로 돌아가 진행
