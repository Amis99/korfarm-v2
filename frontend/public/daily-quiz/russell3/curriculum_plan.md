# 러셀3 (중등 3학년) 365일 일일 퀴즈 교육과정 계획안

## 1. 개요
- **대상**: 중학교 3학년 (러셀3)
- **목표**: 고등 교육과정 연계, 수능 비문학 독해 실전 기초, 통합적 사고력 완성.
  - 교과 심화: 화학 반응, 유전, 별과 우주, 정치/법/경제 심화.
  - 수능 기초: 고난도 비문학 독해(철학/경제/과학기술), 국어 문법(음운 변동/문장 성분/중세 국어).
- **기반 교육과정**: 2022 개정 교육과정
  - **과학**: 화학 반응의 규칙, 기권과 날씨, 운동과 에너지, 자극과 반응, 생식과 유전, 별과 우주.
  - **사회**: 인권과 헌법, 정치 과정, 법, 시장 경제, 국제 경제, 사회 변동.
  - **역사**: 한국 근현대사 흐름 심화, 세계사 현대 부분.
  - **국어**: 문법(음운 변동, 문장 짜임), 논증적 글쓰기, 비판적 듣기/읽기.
- **주제 로테이션**:
  - 1일차: 과학 (화학/생명/지구/물리 - 통합과학 연계)
  - 2일차: 사회 (일반사회 - 통합사회 연계)
  - 3일차: 인문 (철학/현대사 - 비문학 인문 제재)
  - 4일차: 기술 (최신 공학/미디어 기술 - 비문학 기술 제재)
  - 5일차: 예술 (예술 비평/미학 - 비문학 예술 제재)
  - 6일차: 언어 (국어 문법 심화/중세 국어 - 수능 기초)
  - 7일차: 혼합 (고난도 융합 제재/리트 기초)

## 3. 일일 퀴즈 문항 설계 팁 (러셀3 전용)

1. **지문 수준 (Pre-CSAT 심화)**:
   - 16~20문장의 장문 (약 800~1000자).
   - **수능 고난도 제재**: 철학(인식론/윤리학), 경제(환율/금리), 과학(양자역학/유전공학), 기술(AI/블록체인).
   - 정보량이 많고 밀도 높은 글, 도표/그래프 해석이 필수적인 지문.

2. **발문(Stem) 스타일 수능화**:
   - `윗글의 'A'와 <보기>의 'B'를 비교하여 설명한 것으로 가장 적절한 것은?` (3점 문항)
   - `㉠의 관점에서 <보기>의 상황을 평가한 내용으로 적절하지 않은 것은?`
   - `윗글을 바탕으로 추론할 때, <보기>의 ⓐ에 들어갈 말로 적절한 것은?`

3. **문법 강화 (School Grammar)**:
   - 음운 변동 규칙(교체, 탈락, 첨가, 축약)의 완벽한 이해와 적용.
   - 문장의 짜임(안은문장, 이어진문장) 분석 능력.
   - 중세 국어의 특징(훈민정음 언해본 중심) 이해.

4. **소재 활용**:
   - 통합과학/통합사회의 핵심 주제를 미리 다루어 고등 학습 부담 완화.
   - 최신 시사 이슈(기후 위기, AI 윤리, 팬데믹)를 심도 있게 분석.

5. **4지선다 표현력(문장 완성) 예시**:
   - 주제: 역학적 에너지 보존
   - 문장: 공기 저항이나 마찰이 없을 때 운동 에너지와 위치 에너지의 합인 역학적 에너지는 항상 일정하게 보존된다.
   - 빈칸: [공기 저항이나 마찰이 없을 때] -> [운동 에너지와 위치 에너지의 합인] -> [역학적 에너지는] -> [항상 일정하게 보존된다]
