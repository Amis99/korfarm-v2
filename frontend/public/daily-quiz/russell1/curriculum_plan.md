# 러셀1 (중등 1학년) 365일 일일 퀴즈 교육과정 계획안

## 1. 개요
- **대상**: 중학교 1학년 (러셀1)
- **목표**: 교과 개념 심화, 학교 문법 체계화, 비문학 독해력 신장.
  - 교과 심화: 지권/물리/생물 심화, 세계 지리/문화, 정치/경제 기초.
  - 수능 기초: 비문학 독해(인문/사회/과학/기술), 국어 문법(음운/품사/문장).
- **기반 교육과정**: 2022 개정 교육과정
  - **과학**: 지권의 변화, 여러 가지 힘, 생물의 다양성, 기체의 성질, 물질의 상태 변화, 빛과 파동.
  - **사회**: 내가 사는 세계, 다양한 세계 다양한 문화, 자연재해와 안전, 인권과 헌법, 자원과 지속 가능한 삶.
  - **국어**: 문법(품사, 단어 형성), 읽기(예측하며 읽기, 요약하며 읽기), 쓰기(설명하는 글, 감동을 주는 글).
- **주제 로테이션**:
  - 1일차: 과학 (지구과학/물리/화학/생물 - 수능 과학탐구 기초)
  - 2일차: 사회 (지리/일반사회 - 수능 사회탐구 기초)
  - 3일차: 인문 (윤리/철학 - 비문학 인문 제재)
  - 4일차: 기술 (최신 기술/공학 - 비문학 기술 제재)
  - 5일차: 예술 (동서양 미술사/음악 이론 - 비문학 예술 제재)
  - 6일차: 언어 (국어 문법 체계화 - 1학기: 품사/어휘, 2학기: 음운/문장)
  - 7일차: 혼합 (시사/융합/고전 - 고난도 독해)

## 3. 일일 퀴즈 문항 설계 팁 (러셀1 전용)

1. **지문 수준 (Pre-CSAT)**:
   - 12~14문장의 중단문 (약 500~600자).
   - **수능형 제재**: 정보량이 많고 인과관계가 복잡한 과학/기술 지문, 관점이 대립하는 인문/사회 지문.
   - 문단별 소주제 파악 및 글의 전체 구조도(Schema)를 그릴 수 있는 글.

2. **발문(Stem) 스타일 수능화**:
   - `윗글을 바탕으로 <보기>를 이해한 내용으로 적절한 것은?` (3점 문항 유형)
   - `㉠과 ㉡을 비교한 내용으로 가장 적절한 것은?`
   - `윗글의 글쓴이가 <보기>의 주장에 대해 보일 반응으로 적절한 것은?`

3. **문법 강화 (School Grammar)**:
   - 중1 교과서 핵심 문법(품사, 단어 형성, 음운 체계)을 완벽하게 마스터.
   - 예시 중심의 탐구형 문법 문제 (암기 지양).

4. **소재 활용**:
   - 중학교 1학년 전 과목 교과 개념을 수능 비문학 제재로 가공.
   - 시사 쟁점(기본권, 환경, AI)을 교과 개념과 연결.

5. **4지선다 표현력(문장 완성) 예시**:
   - 주제: 훅의 법칙 (탄성력)
   - 문장: 용수철의 늘어난 길이는 작용한 힘의 크기에 비례하여 증가한다.
   - 빈칸: [용수철의 늘어난 길이는] -> [작용한 힘의 크기에] -> [비례하여] -> [증가한다]
