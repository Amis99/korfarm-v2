# 프레게1 (초등 4학년) 365일 일일 퀴즈 교육과정 계획안

## 1. 개요
- **대상**: 초등학교 4학년 (프레게1)
- **목표**: 통합적 사고력 확장, 민주 시민 의식 함양, 과학적 탐구 능력 심화.
- **기반 교육과정**: 2022 개정 교육과정
  - **과학**: 지층과 화석, 식물의 한살이, 물체의 무게, 혼합물의 분리, 화산과 지진, 물의 여행.
  - **사회**: 지역의 위치와 특성, 우리가 알아보는 지역의 역사, 공공 기관과 주민 참여, 경제 생활과 바람직한 선택.
  - **국어**: 생각과 느낌을 나누어요, 내용을 간추려요, 사전에 없는 낱말, 전기문 읽기.
- **주제 로테이션**:
  - 1일차: 과학 (지구, 물질, 생명 - 탐구 중심)
  - 2일차: 사회 (지리, 정치, 역사 - 조사 중심)
  - 3일차: 인문 (인물, 가치, 고전 - 독서 중심)
  - 4일차: 기술 (소프트웨어, 로봇, 발명 - 창의 중심)
  - 5일차: 예술 (음악, 미술, 체육/보건 - 감상 및 실천)
  - 6일차: 언어 (어휘력, 문법, 독해 - 논리 중심)
  - 7일차: 혼합 (시사, 상식, 융합 - 종합적 사고)

## 3. 일일 퀴즈 문항 설계 팁 (프레게1 전용)

1. **지문 수준**:
   - 6~8문장의 완성된 문단.
   - 설명문, 논설문, 전기문, 기사문 등 비문학 텍스트 비중 확대.
   - 문단 간의 연결 고리(접속사 등)를 파악해야 하는 글.

2. **발문(Stem) 스타일**:
   - "윗글의 내용과 일치하지 않는 것은?", "글쓴이가 주장하는 바는?" 등 수능형 발문의 기초.
   - `<보기>`를 활용한 적용형 문제, 자료 해석형 문제.

3. **선택지(Choice)**:
   - 문장형 선택지 기본.
   - 오답 매력도(Distractor) 강화 (지문에 언급되었으나 인과관계가 틀린 것 등).

4. **소재 활용**:
   - 4학년 교과 개념(지층, 화산, 지역 사회, 민주주의)을 적극 활용.
   - 시사적인 문제(환경, 인권)를 초등 수준에서 다룸.

5. **4지선다 표현력(문장 완성) 예시**:
   - 주제: 화산 활동
   - 문장: 화산이 분출할 때 나오는 용암, 화산 가스, 화산재 등을 화산 분출물이라고 한다.
   - 빈칸: [화산이 분출할 때 나오는] -> [용암, 화산 가스, 화산재 등을] -> [화산 분출물이라고] -> [한다]
