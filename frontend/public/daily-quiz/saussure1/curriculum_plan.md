## 3. 일일 퀴즈 문항 설계 팁 (소쉬르1 전용)

1. **지문 수준**:
   - 2~3문장의 아주 짧은 글.
   - 구어체(~해요) 사용 권장.
   - 어려운 한자어는 피하고 쉬운 고유어 사용.

2. **발문(Stem) 스타일**:
   - "~무엇인가요?", "~찾아보세요." 등 친절한 어투.
   - 직관적인 질문.

3. **선택지(Choice)**:
   - 단답형 단어 위주.
   - 문장형 선택지일 경우 10어절 이내로 짧게.

4. **소재 활용**:
   - 아이들에게 친숙한 '학교', '집', '놀이터', '마트' 배경.
   - 동물, 식물 의인화 가능 (예: 다람쥐가 말했어요).

5. **4지선다 표현력(문장 완성) 예시**:
   - 주제: 학교 가는 길
   - 문장: 횡단보도에서는 손을 들고 건넙니다.
   - 빈칸: [횡단보도에서는] -> [손을] -> [들고] -> [건넙니다]
